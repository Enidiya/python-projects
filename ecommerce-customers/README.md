# Ecommerce Customers Analysis

## Project Overview
This project analyzes ecommerce customer data to understand customer behavior and spending patterns using Python.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
The dataset contains customer-level ecommerce data used for exploratory data analysis.

## Key Insights
- Relationship between annual income and spending
- Customer segmentation patterns
- Spending trends across different demographics
